Cloning to a specific folder:
git clone <repo> <folderName>

Cloning a specific branch of a repository:
git clone -branch <branchName> <repo>

After Changes in git folder

git add File Name
for name with spaces use git add 'file name.jpg' 
git commit
git commit -m "Message to be Displayed"
git push

git branch
List all of the branches in your repo. Add a <branch> argument to create a new branch with the name <branch>.
git checkout -b <branch>
Create and check out a new branch named <branch>.
Drop the -b flag to checkout an existing branch.
git merge <branch>
Merge <branch> into the current branch.
